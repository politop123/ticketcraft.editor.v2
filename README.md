# TicketCraft Editor v2

Production-ready ticket template editor module for Nuxt 4 + Vue 3 + TypeScript.

## Quick start

```bash
npm install
npm run dev
```

Open: `http://localhost:3000/ticket-editor`

## Scripts

- `npm run dev` — local development
- `npm run build` — production build
- `npm run preview` — preview production build

## Notes

- Includes Pinia-based editor state.
- Exports template JSON + rendered HTML.
- Autosave draft is stored in browser `localStorage`.
